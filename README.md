# DatascrapeCollections
In this repository, I put my previous data scrape code together. While it may seem somewhat outdated by today's technological standards, it did incorporate specific handling mechanisms designed to recognize various web applications prevalent at the time.

HitinBaidu: 
1. Scraping Baidu Hot Trends and Weibo Hot Treand by the keyword and date
2. TF-IDF analysis
3. Kmeans analysis
4. Sentimental analysis based on the words

Xiaohongshu&Weibo:
1. Scraping Weibo and Xiaohongshu based on the keyword and date

Taobao:
1. Scraping Taobao Item list based on the keyword and date
2. Jump over the automatic test by the automatic slider
